Annotation Instructions

All nodes in the diagram must be labeled with one of these 4 categories. To do this, click the corresponding button and then click on the node.


Sufficiency Nodes that must be present in the diagram to answer the user's query
Completeness Nodes that don't have to be in the diagram, but whose presence is helpful for understanding
Hallucinations Incorrect or unjustifiable (fabricated and irrelevant) nodes
Verbosity Superfluous nodes: entities from the code that should not be in the diagram answering the query

If the query mentions a specific entity, you can confidently mark it as Sufficiency.

How to distinguish Sufficiency from Completeness: imagine that the node in question is absent from the diagram. Does the diagram still provide enough information for a direct answer? If yes, then it's Completeness; if something essential to the answer's core meaning is missing, then it's Sufficiency.
